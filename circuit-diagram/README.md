## Circuit Diagram

This image shows the complete wiring of the Arduino-based password door lock system.
It includes connections for the 16×2 LCD display, 4×4 matrix keypad, and servo motor.

The LCD is interfaced in 4-bit mode, the keypad is connected using row-column
matrix wiring, and the servo motor is controlled through a PWM signal pin.
Proper power and ground connections are ensured for stable operation.
